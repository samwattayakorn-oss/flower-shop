====== Purpose ======

กำหนดความหมายของ "Meaning" ภายในระบบ

====== Core Meaning ======

Meaning คือคุณค่าที่เกิดจากการตีความข้อมูลภายใต้เงื่อนไขหนึ่ง

====== Condition ======

Meaning ต้องอาศัย Context

Meaning ต้องสามารถอธิบายได้

Meaning ต้องเป็นไปตามมาตรฐาน UKS


====== Principle ======

Data ไม่มี Meaning

Meaning เกิดจาก Relation

Relation เกิดจาก Context

Context เกิดจาก Condition
