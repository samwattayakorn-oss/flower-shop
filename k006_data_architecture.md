Data Architecture

====== Purpose ======

กำหนดมาตรฐานของข้อมูลภายในระบบ

====== Core Meaning ======

Data คือหน่วยข้อมูลพื้นฐานที่ใช้สร้างความหมาย ความสัมพันธ์ และผลลัพธ์ของระบบ

====== Principle ======

Data ต้องมีความถูกต้อง

Data ต้องสามารถอ้างอิงได้

Data ต้องสามารถนำกลับมาใช้ซ้ำได้

Data เป็นพื้นฐานของ Knowledge

====== Structure ======

Raw Data

↓

Information

↓

Meaning

↓

Knowledge

↓

Decision

↓

Result

====== Condition ======

Data ต้องมีโครงสร้างชัดเจน

Data ต้องสามารถตรวจสอบได้

Data ต้องรองรับการขยายในอนาคต

Data ต้องสอดคล้องกับมาตรฐาน UKS

====== Relation ======

Website

กำหนดโครงสร้างข้อมูลทุกประเภท
