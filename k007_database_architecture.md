Database Architecture

====== Purpose ======

กำหนดมาตรฐานของฐานข้อมูล

====== Core Meaning ======

Database คือพื้นที่จัดเก็บข้อมูลอย่างเป็นระบบ เพื่อให้ค้นหา แก้ไข และเชื่อมโยงข้อมูลได้อย่างมีประสิทธิภาพ

====== Principle ======

Database ต้องจัดเก็บข้อมูลอย่างเป็นระบบ

Database ต้องรองรับการค้นหาและเชื่อมโยงข้อมูล

Database ต้องสามารถขยายได้

Database ต้องรองรับหลายประเภทของข้อมูล

====== Condition ======

ข้อมูลต้องมีความถูกต้อง

ข้อมูลต้องไม่สูญหาย

Database ต้องรองรับการสำรองข้อมูล

Database ต้องเป็นไปตามมาตรฐาน UKS
