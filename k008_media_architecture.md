====== Purpose ======

กำหนดมาตรฐานของไฟล์สื่อ

====== Principle ======

Media ต้องจัดเก็บอย่างเป็นระบบ

Media ต้องสามารถอ้างอิงได้

Media ต้องรองรับการนำกลับมาใช้ซ้ำ

Media ต้องรองรับหลายรูปแบบไฟล์
