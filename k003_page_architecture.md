Page Architecture

====== Purpose ======

กำหนดมาตรฐานของหน้าเว็บไซต์

====== Core Meaning ======

Page คือการรวม Module หลายตัวเพื่อสร้างหน้าที่สมบูรณ์

====== Principle ======

Page ต้องประกอบจาก Module

Page ต้องสามารถปรับเปลี่ยน Module ได้โดยไม่กระทบโครงสร้างหลัก

Page ต้องรองรับการขยายในอนาคต

Page ต้องสอดคล้องกับมาตรฐาน Website

====== Structure ======

Header

↓

Content

↓

Module

↓

Footer

====== Condition ======

ทุก Page ต้องมีวัตถุประสงค์ชัดเจน

ทุก Module ภายใน Page ต้องทำงานร่วมกันได้

Page ต้องสามารถเชื่อมโยงกับ Website ได้

Page ต้องเป็นไปตามมาตรฐาน UKS

====== Relation ======

Page ใช้ Module

Module ใช้ Library

Library ใช้ Knowledge

====== Example ======

Home Page

Product Page

Category Page

Contact Page

About Page

Dashboard Page

====== Scope ======

อธิบายเฉพาะโครงสร้างของหน้าเว็บ

====== Future Expansion ======

รองรับ Website ทุกประเภท
